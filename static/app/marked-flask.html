Marked-Flask

Flask app that renders md to html by marked.js 


'''thanks for trying marked-flask😊, it's a simple way to read your .md online🍦'''

marked-flask是一个基于marked.js渲染md文件的flask应用

简介

该应用基于flask，利用ajax读取本地markdown文件，在前端使用marked.js直接渲染为html文本。配合github-markdown-css和highlight.js实现对文本内容和代码块的特殊渲染

本站点的在线文档即使用marked-flask框架

下载

使用git克隆源码仓库

$ git clone https://github.com/Landers1037/marked-flask.git
